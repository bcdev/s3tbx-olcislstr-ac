SNAP Data Processors - OLCI Atmospheric Correction Algorithm Specification

Algorithm Specification

The underlying algorithm for the MERIS atmospheric correction scheme has been developed by University of Swansea and Free University of Berlin and is described in great detail in [1]. For this processor the algorithm has been 1:1 transferred to OLCI in order to provide OLCI SDRs at the 15 MERIS heritage wavelengths. The current processor does not yet provide SDRs for the new OLCI wavelengths.
